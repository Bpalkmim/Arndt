Encontrar Vagas

O usuário motorista deve inicializar o aplicativo e pedir que o sistema encontre vagas próximas a ele. Então o sistema as procura no banco de dados e informa o usuário.

Aplicativo para "smartphone" para o motorista e sevidor WEB.

Usuário motorista

Usuário motorista
    Objetivo: Encontrar vagas próximas a ele de maneira fácil.
Aplicativo móvel
    Objetivo: Obter a localização do "smartphone" e informar ao servidor.
GPS
    Objetivo: Calcular a localização do "smartphone".
Servidor WEB
    Objetivo: Acessar a base de dados e obter as vagas disponíveis próximas ao usuário.
Base de dados de disponibiliade de vagas
    Objetivo: Armazenar a localização das vagas e informações sobre sua disponibilidade. Além disso deve ser capaz de encontrar vagas dada uma localização próxima.
